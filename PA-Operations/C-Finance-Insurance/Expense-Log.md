# 💰 Expense Log

> Record all expenses incurred for Boss or on his behalf. Reconcile after each trip.

---

## Expense Log Fields

| Field | Type | Notes |
|---|---|---|
| Date | Date | Date expense was incurred |
| Category | Select | See categories below |
| Description | Text | Brief description |
| Amount | Number | In original currency |
| Currency | Select | MYR / EUR / GBP / BRL / USD / IDR |
| Amount (MYR) | Number | Converted to MYR |
| Exchange Rate Used | Number | Rate at time of transaction |
| Card Used | Text | Last 4 digits of card |
| Booked By | Select | Boss / PA |
| Linked Trip | Relation | → Trips database |
| Receipt | File | Upload receipt or photo |
| Reimbursable | Checkbox | Needs to be reimbursed? |
| Status | Select | Pending / Reconciled / Reimbursed |
| Notes | Text | |
