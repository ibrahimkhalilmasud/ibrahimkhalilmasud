## ✅ Seat Booking Checklist

### 1. Seat Selection
- [ ] Log in to airline website / app using loyalty program account
- [ ] Select preferred seat:
  - Window seat preferred (long-haul)
  - Extra legroom if available (especially for 67-year-old senior traveller)
  - Avoid seats near toilets or galley where possible
- [ ] Confirm seat number: **{SEAT_NUMBER}**
- [ ] Take screenshot of seat map / confirmation

### 2. Meal Preference
- [ ] Set meal preference to **healthy / low-fat / special meal**
- [ ] Confirm meal type selected: **{MEAL_TYPE}**
- [ ] Note: Special meals are usually served first and tend to be fresher
- [ ] Screenshot confirmation

### 3. Lounge Access
- [ ] Confirm lounge access eligibility:
  - Qatar Silver: Priority Pass / Qatar Lounge access on select routes
  - KrisFlyer: SilverKris Lounge (Business/higher tier)
  - BA Executive Club: Paid / partner lounge access
- [ ] Check lounge location at airport: **{LOUNGE_NAME}** at **{TERMINAL}**
- [ ] Confirm lounge opening hours: **{HOURS}**
- [ ] Note: Qatar Silver may have lounge access on Qatar-operated flights

### 4. Check-In Reminder
- [ ] Note online check-in open date: **{DATE}** (usually 24–48 hours before departure)
- [ ] Set Notion/Calendar reminder for check-in open
- [ ] Set Calendar reminder for airport departure time (allow 3 hours for international)

### 5. Baggage
- [ ] Confirm baggage allowance: **{ALLOWANCE} kg checked / {CARRY_ON} kg carry-on**
- [ ] Note any excess baggage fees if needed
- [ ] Confirm loyalty tier baggage bonus (Qatar Silver: +1 bag on some routes)

### 6. Transit Visa (if applicable)
- [ ] Layover country: **{LAYOVER_COUNTRY}**
- [ ] Layover duration: **{HOURS}**
- [ ] Transit visa required? **{YES/NO}**
- [ ] If yes: obtain transit visa before travel

### 7. Final Confirmation
- [ ] Send Boss flight confirmation via WhatsApp (use Template 2)
- [ ] Update Flights database — mark "Seat Checklist Done: ✅"
- [ ] File booking confirmation in Google Drive / Proton Drive
