# 🌍 Trips Master Database

> Master record for every trip. Link all flights, hotels, and expenses here.

---

## Database Fields

| Field | Type | Notes |
|---|---|---|
| Trip Name | Title | e.g. "Europe Winter 2025" |
| Destinations | Multi-select | All countries/cities visited |
| Departure Date | Date | From KL (or origin) |
| Return Date | Date | Back to KL (or home base) |
| Duration (days) | Formula | Auto-calculated |
| Status | Select | Planning / Upcoming / Active / Completed |
| Travellers | Multi-select | Boss / Wife / Both |
| Linked Flights | Relation | → Flights database |
| Linked Hotels | Relation | → Hotels database |
| Total Budget (MYR) | Number | Planned total |
| Total Actual (MYR) | Number | Filled after trip |
| Variance (MYR) | Formula | Actual − Budget |
| Visas Required | Text | List countries needing visa |
| Pre-Trip Report | Checkbox | Prepared? |
| Post-Trip Report | Checkbox | Completed? |
| Notes | Text | Key highlights, changes |
